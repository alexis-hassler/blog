private Event messageEvent;

Ça permet d'envoyer des événements types messages :
messageEvent.fire( new Message("hello") )

On peut aussi qualifier les événements qu'on envoie :
messageEvent
    .select(new AnnotationLiteral() {})
    .fire( new Message("hello") )

Ça se complique pour les qualifiers qui ont des attributs. Il faut créer une classe qui hérite de AnnotationLiteral et qui implémente le qualifier :

public class GreetingAnnotationLiteral extends AnnotationLiteral implements Greeting {
   ...
}


Attention, dans une extension, pas d'@Inject => utiliser le BeanManager
